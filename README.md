# Recharge Records Bot & Dashboard

A complete solution for managing recharge records via Telegram bot and web dashboard.

## Features

- 🤖 Telegram bot for uploading Excel files
- 📊 Dashboard with search, edit, delete, add functionality
- 💾 MongoDB storage with duplicate handling
- 🔐 Secure login protection
- 📈 Statistics overview
- 📤 Export to CSV
- 📱 Mobile responsive
